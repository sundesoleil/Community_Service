# Community Service
여러 가지 카테고리가 있는 커뮤니티 서비스 

# 기획의도
CRUD 기능에 대한 이해

# 개발 환경
Java, Spring Boot, MySQL, Ajax, mybatis, Jquery...

# 구현 기능
* 로그인/회원가입
* 글쓰기/수정/삭제
* 댓글
* 좋아요
* 조회수 반영
